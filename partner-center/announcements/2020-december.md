## <a name="an-update-of-the-us-microsoft-365-business-voice-with-calling-plan-offer-is-coming-soon"></a><a name="2"></a>通話プラン サービスのある米国 Microsoft 365 Business Voice の更新が近日公開予定

### <a name="categories"></a>Categories

- 日付:2020 年 12 月 3 日
- 製品サービス/市場

### <a name="summary"></a>まとめ 

Microsoft は、2021 年 1 月 1 日に、米国のパートナー様とお客様の、通話プラン サービスのある新しい Microsoft 365 Business Voice への移行を開始します。 パートナー側での操作は必要ありません。

### <a name="impacted-audience"></a>影響を受ける対象ユーザー 

クラウド ソリューション プロバイダー (CSP) プログラムを通じて取引を行う、すべてのパートナー様

### <a name="details"></a>詳細

Microsoft は、米国内の通話プラン サービスのある既存の Business Voice を新しいオファーに置き換えて、通信製品の社内要件をサポートします。 新しいオファーの機能セットと価格は変わりません。 オファーの変更は Microsoft 内部のものであり、CSP パートナー様が通話プラン サービスのある Business Voice をマーケティング、販売、サポートする方法に影響を与えることはありません。 変更は、このオファーにのみ適用されます。

新規販売では、2021 年 1 月 1 日から始まる新しいオファーを使用します。 古いオファーは、価格表で新しいオファーに置き換えられます。

既存のお客様は、更新日に新しいオファーに自動的に移行されます。 パートナー様や新規のお客様が対応を行う必要はありません。

   |Microsoft 365 Business Voice (US)|86713ec1-ad33-42cf-a1ce-8397d4d875fe|PZW-00023|
   
** 古いオファーの詳細

   |**プラン名**|**プラン ID**|**マテリアル ID**|

## <a name="now-live-partner-center-api-updates-and-user-interface-enhancements-for-the-education-customer-validation-process"></a><a name="1"></a>公開中:教育機関の顧客の検証プロセスにおけるパートナー センター API の更新とユーザー インターフェイスの機能強化

### <a name="categories"></a>Categories

- 日付:2020 年 12 月 3 日
- 機能

### <a name="impacted-audience"></a>影響を受ける対象ユーザー 

クラウド ソリューション プロバイダー プログラムを通じてアカデミック向けオファーを販売しているパートナー様

### <a name="summary"></a>まとめ 

Microsoft の基盤は信頼です。 Microsoft は、クラウド ソリューション プロバイダー プログラムにおけるアカデミック向けオファーの取引を行うにあたって顧客を検証するための、法令に準拠した安全な手段の提供に努めています。 このコミットメントの一環として、顧客の審査プロセスの明瞭性と可視性が向上し、より正確なデータを入力できるようになるパートナー センター API とユーザー インターフェイスの機能強化を導入する予定です。これにより、顧客審査の成功率が上がります。 

**パートナー センターの機能強化** 

- 正確なデータ入力をサポートし、Microsoft による教育機関顧客の検証プロセスを改善するための新しい GET および POST Qualifications API。 

- 正確なデータ入力をサポートし、Microsoft による教育機関顧客の検証プロセスを改善するためのユーザー インターフェイスの拡張機能。 

**既存の GET および PUT 認定 API の非推奨** 

既存の GET および PUT 認定 API は、**2021 年 2 月末まで** に廃止されます。 教育機関向け特価プランを購入できるようにするには、その時点で、新しい GET および POST パートナー センター API に移行している必要があります。  

**テスト** 

顧客検証の成功に必要な API とデータ入力について理解を深めるために、パートナー様はこれらの拡張機能をテストすることができます。 テストへの参加を希望するパートナー様は、[パートナー センターの教育機関の顧客向けテスト ガイド](https://partner.microsoft.com/resources/detail/partner-center-edu-testing-guide-pdf)をダウンロードして、事前の準備やサインアップの方法、およびテスト段階で行うことに関する情報をご覧ください。

**図書館と博物館の顧客** 

上記の拡張機能に加えて、Microsoft は、CSP オファーを取り引きできる教育機関の顧客を拡大する、図書館および博物館の顧客向けアカデミック価格のオファーを提供できるようになることをお知らせいたします。 

Microsoft では、すべての顧客または提案された顧客の適格教育ユーザーとしての状態をレビューする権利を留保します。 詳細については、[アカデミック適格教育ユーザーの要件](https://www.microsoftvolumelicensing.com/DocumentSearch.aspx?Mode=3&DocumentTypeId=7)をご覧ください。
