## <a name="now-available-windows-7-year-2-esu-for-cloud-solution-provider-csp"></a><a name="6"></a>クラウド ソリューション プロバイダー (CSP) 向けの Windows 7 Year 2 ESU が利用可能に

### <a name="categories"></a>［カテゴリ］

- 日付:2020-06-29
- 次のように入力します。製品サービス/市場
- 状態:提供中

### <a name="summary"></a>要約

Windows 7 の Year 2 ESU (拡張セキュリティ更新プログラム) の価格を、[リソース ギャラリー](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)で確認できます。

### <a name="impacted-audience"></a>対象

クラウド ソリューション プロバイダー (CSP) プログラムに参加しているすべてのパートナー

### <a name="details"></a>詳細情報

Windows 7 のサポートは 2020 年 1 月 14 日に終了しました。 多くの顧客は Windows 10 を展開しようとしているところですが、Microsoft はアップグレードの進捗状況が顧客によって異なることを理解しています。 Windows 7 の Year 1 ESU の有効期間は、2021 年 1 月 12 日に終了します。 顧客は次の年の計画を立て始めるところでしょう。  

2021 年も Windows 7 デバイスの使用を継続するつもりの顧客を対象として、Windows 10 に関する決定を行うことができるように、Windows 7 Year 2 ESU の価格をクラウド ソリューション プロバイダー (CSP) に対して公開します。 詳細については、[リソース ギャラリー](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)をご覧ください。

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>パートナー センターのリレーションシップ削除機能の更新

### <a name="categories"></a>［カテゴリ］

- 日付:2020-06-29
- 次のように入力します。機能
- 状態:提供中
- 
### <a name="summary"></a>要約

2020 年 6 月 29 日以降、クラウド ソリューション プロバイダー (CSP) の間接プロバイダーと間接リセラーは、パートナー センターを使用してダウンストリームのリレーションシップを削除できるようになりました。そうするためにパートナー センターのサポート チケットを提出する必要はありません。 これにより、パートナーはこれらの操作を行う際の時間をかなり節約することができます。

### <a name="impacted-audience"></a>対象

クラウド ソリューション プロバイダー (CSP) の間接プロバイダーおよび間接リセラー

### <a name="scenarios-supported"></a>サポートされるシナリオ:

- 間接リセラーとの確立されたリレーションシップを持つ間接プロバイダーは、パートナー センターを使用してそのリレーションシップを削除することができます。
- エンド カスタマーとの確立されたリレーションシップを持つ間接リセラーは、パートナー センターを使用してそのリレーションシップを削除することができます。

上記のシナリオでの操作は、パートナー センターの管理エージェントが行う必要があります。

### <a name="next-steps"></a>次のステップ:

- 詳細については、[リソース ギャラリー](https://partner.microsoft.com/resources/collection/partner-center-relationship-removal#)をご覧ください。
- オペレーション資料ギャラリー コレクションにある資料を確認:パートナー センターのリレーションシップ削除機能の更新。
- この情報を組織内で共有して、適切なチームがプロセスを確認して更新できるようにしてください。

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>請求頻度の既定値が年間から月単位に変更される

### <a name="categories"></a>［カテゴリ］

- 日付:2020-06-17
- 次のように入力します。機能
- 状態:近日提供予定

### <a name="summary"></a>要約

パートナー センターの購入エクスペリエンスで、請求頻度の既定値が年間から月単位に変更されます。

### <a name="impacted-audience"></a>対象

クラウド ソリューション プロバイダー (CSP) の直接プロバイダーおよび間接プロバイダー

### <a name="details"></a>詳細情報

請求頻度の設定によって、パートナー様が購入したサービスに対する請求頻度が決まります。 パートナー様は、パートナー センター カタログで購入するときに、**レビュー**のステップでこの設定を確認できます。 この設定の既定値は、ライセンスベースのサービスに年間請求が導入されて以来、**年間**になっていました。 パートナー様からいただいたフィードバックに基づいて、パートナー センター チームでは 6 月 17 日から、この既定値を**年間**から**月単位**に変更します。これにより、パートナー様のご要望に一層合致したものになります。 パートナー様は、トランザクションを送信する前に [レビュー] ページで、またはトランザクションの後に [[サブスクリプションの管理] ページ](../billing-annual-monthly.md#change-billing-frequency)で、これを変更できます。 なお、年間請求のみに対応しているプランについては、引き続き年間が既定値になります。 この既定値の変更は、パートナー センター ポータルのユーザー エクスペリエンスにのみ適用されます。パートナー センター API の機能に変更はありません。

この変更によって、パートナー様が年間請求のサブスクリプションを誤って購入する可能性が低くなります。

Microsoft 365 F1 は新しいサービスに更新されます。 新しいプランは 6 月 15 日にリリースされ、以前の F1 と置き換わります。

元の F1 プラン:

   |**プラン名**|**プラン ID**|**マテリアル ID**|**プロビジョニング ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|1ce4a4ba-65f7-49e7-bb28-d92d07d58b86|1PI-00005|M365_F1|
   |Microsoft 365 F1 Trial|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

新しい F1 サブスクリプションを購入するパートナー様は、6 月 15 日以降、以下のプラン ID を使用する必要があります。 パートナー様は、パートナー センター カタログでこれらの ID を確認できます。 パートナー センター API を利用するパートナー様は、以下の情報を使用することもできます。

   |**プラン名**|**プラン ID**|**マテリアル ID**|**プロビジョニング ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|17b4a1d6-2c65-478e-a046-51cf40d25d76|1PI-00012|M365_F1_COM|
   |Microsoft 365 F1 Trial|2e4e9424-06db-4d35-b2d8-3a1b0ef1dc63|1PI-00013|M365_F1_COMM|

### <a name="call-to-action"></a>行動喚起
現在の Microsoft 365 F1 製品を使用しているパートナー様には、何の対処も必要ありません。 Exchange Kiosk および MS Search のサービス プランは、6 月の終わりまでに既存のサービス プランに追加されます。

6 月 15 日以降に Microsoft 365 F1 を購入するパートナー様は、適切なプランを確実に購入するため、新しい F1 プラン情報をご使用ください。 新しい F1 プランの ID とその他の情報は、7 月 1 日に更新される 7 月のクラウド ソリューション プロバイダー (CSP) 価格表で更新されます。

## <a name="new-and-end-of-life-microsoft-dynamics-365-and-power-platform-offers-and-products"></a><a name="1"></a>Microsoft Dynamics 365 および Power Platform の新規および終了となるプランと製品

### <a name="categories"></a>［カテゴリ］

- 日付:2020-06-11
- 次のように入力します。製品サービス/市場
- 状態:近日提供予定

### <a name="summary"></a>要約

2020 年 7 月 1 日に、Microsoft では、Dynamics 365 および Power Platform の新しいプランと製品をリリースし、該当する Fraud Protection 製品を廃止します。

### <a name="impacted-audience"></a>対象

- Microsoft Dynamics 365 Fraud Protection – Loss Prevention:この製品を使用すると、小売業者は、従来型の店舗での在庫減少を回避することができます。
- Microsoft Dynamics 365 Fraud Protection – Purchase Protection:この製品を使用すると、マーチャントは、不正なアクティビティを検出して防止することで、オンライン トランザクションを保護することができます。
- Microsoft Dynamics 365 Fraud Protection – Account Protection:この製品を使用すると、マーチャントは、不正なアクティビティを検出して防止することで、オンライン顧客アカウントを保護することができます。

販売終了となる Microsoft Dynamics 365 Fraud Protection 製品:

Microsoft Dynamics 365 Fraud Protection の次のプランは、2020 年 7 月 1 日の価格表から削除されます。

- Microsoft Dynamics 365 Fraud Protection
- Microsoft Dynamics 365 Fraud Protection Addl Assessments
