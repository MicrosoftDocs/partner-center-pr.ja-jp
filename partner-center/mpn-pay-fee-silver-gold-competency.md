# <a name="how-to-pay-the-fee-associated-with-a-silver-or-gold-competency"></a>シルバーまたはゴールドコンピテンシーに関連する料金を支払う方法

**適切なロール**

- MPN パートナー管理者
- MPN アカウント管理者

この記事では、シルバーまたはゴールド Microsoft Partner Network コンピテンシーの購入に関連する料金の支払い方法とタイミングについて説明します。 この記事では、ご希望の支払い方法を変更する方法、注文を取り消す方法、シルバーからゴールドコンピテンシーへの移行時に25% の料金割引を受ける方法についても説明します。

> [!NOTE]
> シルバーまたはゴールドのコンピテンシーを取得する利点、特定のコンピテンシー、および固有の要件について詳しくは、「 [Microsoft Partner Network コンピテンシー](https://partner.microsoft.com/membership/competencies)」をご覧ください。 また、Microsoft Partner Network [コンピテンシーのライセンステーブル] を確認して、利用可能な各コンピテンシーの特典の詳細を参照することもできます。

## <a name="meet-a-competencys-requirements-and-pay-the-fee"></a>コンピテンシーの要件を満たし、料金を支払う

特定のシルバーまたはゴールドコンピテンシーを達成するための要件を既に満たしている場合 (または、既にシルバーまたはゴールドコンピテンシーがある場合) は、コンピテンシーの料金をすばやく確認できます。 パートナーセンターで [ [比較-オファー](https://partner.microsoft.com/membership/compare-offers) ] タブを選択するだけです。

それ以外の場合は、次の手順に従って、シルバーまたはゴールドコンピテンシーの料金を支払います。 この手順では、次の前提条件を満たしていることを前提としています。

- お客様は既に Microsoft Partner Network に [参加](https://partner.microsoft.com/membership) しているパートナーです。

- 特定のシルバーまたはゴールドコンピテンシーを得るための [要件](https://partner.microsoft.com/membership/competencies) を既に満たしています。

- [グローバル管理者または MPN partner admin](https://docs.microsoft.com/partner-center/permissions-overview)のパートナーセンターロールが割り当てられています。これらのロールのいずれかを使用していないゲストユーザーの場合は、以下で説明するパートナーセンターの一部のページへのアクセスが拒否されます。 この問題が発生した場合は、 [グローバル管理者に連絡](https://docs.microsoft.com/partner-center/find-your-role) して、適切なロールを割り当てるように依頼してください。 

1. 組織の職場アカウントの資格情報を使用して、パートナーセンターの [ダッシュボード](https://partner.microsoft.com/dashboard)にサインインします。

2. ナビゲーションメニューから [ [メンバーシップオファー](https://partner.microsoft.com/dashboard/mpn/offers) ] を選択します。 

3. [ **購入可能なプラン**] で、特定のコンピテンシーに関連付けられているシルバーメンバーシップ * * タイルまたは **Gold メンバーシップ** タイルを見つけます。 

4. 特定のシルバーまたはゴールドメンバーシップタイルの右下にある [ **サブスクライブ** ] オプションを選択します。

   :::image type="content" source="images/offers/1-mpn-competency-payment-offers-available-for-purchase.png" alt-text="パートナーがサブスクライブまたは購入できるシルバーまたはゴールドのメンバーシップを示すメンバーシップのページ。" border="false":::

   > [!Note]
   > 必要なシルバーまたはゴールドコンピテンシーの要件をまだ満たしていない場合、シルバー/ゴールドのメンバーシップをサブスクライブしたり、関連するコンピテンシー料金を支払いたりすることはできません。 また、既に gold コンピテンシーパートナーである場合は、シルバーコンピテンシーの増分特典が自動的に授与されます。 このシナリオでは、シルバーコンピテンシー料金を支払うオプションも使用できません。

4. [ **購入の詳細** ] ページで、詳細を確認して確認します。

   (会社 [の **税 ID** 情報の](https://docs.microsoft.com/partner-center/organization-tax-info) 更新、または会社の [**請求書** から会社の住所への販売](https://docs.microsoft.com/partner-center/mpn-get-action-pack)の詳細については、こちらを参照してください。

   :::image type="content" source="images/offers/2-mpn-competency-payment-preference.png" alt-text="メンバーシップには、支払いの設定を含む購入詳細ページが用意されています。" border="false":::

5. ページの左下にある **[支払い設定の選択** ] セクションを探します。 [ **クレジットカード** ] オプションまたは [ **電子銀行転送** ] オプションのいずれかを選択し、[ **次へ**] を選択します。 いずれかのオプションに関連する詳細な手順については、次のセクションを参照してください。

### <a name="use-a-credit-card-as-your-payment-preference"></a>支払い設定としてクレジットカードを使用する

関連するシルバーまたはゴールドコンピテンシーメンバーシップのクレジットカードの支払いを完了するには、次の手順に従います。

1. お支払い方法として **クレジットカード** を選択した場合は、クレジットカードの詳細ページが表示されます。 

   :::image type="content" source="images/offers/3-mpn-competency-payment-credit-card-details.png" alt-text="メンバーシップは、支払いクレジットカードの詳細ページを提供します。" border="false":::

2. このページの上部にある [ **契約** ] リンクを選択し、会社に代わって MPN プランの購入条件を確認します。 本ライセンス条項は、請求先アカウント、更新、返金資格、および税金に関する重要な情報を提供します。

3. クレジットカードの詳細と課金情報を関連フィールドに入力し、[ **支払い**] を選択します。

   お支払いは、サードパーティのセキュリティで保護された支払いゲートウェイを通じて収集されます。

   > [!NOTE]
   > このページで入力したアドレスが、クレジットカードのアドレスに対応していることを確認してください。 有効なエンティティアドレスがクレジットカードの住所と異なる場合は、このクレジットカードの詳細ページで法人の住所を使用しないでください。 また、郵便番号が正しいことと、このページで入力したアドレスに対応していることを確認してください。

クレジットカード情報を受信し、トランザクションが完了するとすぐに、シルバーまたは gold コンピテンシーのサブスクリプションが直ちにアクティブ化されます。

### <a name="use-electronic-bank-transfer-as-your-payment-preference"></a>支払い設定として電子銀行転送を使用する

次の手順に従って、関連付けられているシルバーまたはゴールドコンピテンシーのメンバーシップに関する電子銀行の譲渡を完了します。

1. お支払い方法として [ **電子銀行転送** ] を選択した場合は、[ **購入の詳細の確認** ] ページが表示されます。

   :::image type="content" source="images/offers/4-mpn-competency-payment-review-purchase-details.png" alt-text="メンバーシップは、電子銀行転送の購入の詳細ページを提供します。" border="false":::

1. このページの下部にある [ **契約** ] リンクを選択し、会社に代わって MPN プランの購入条件を確認します。 本ライセンス条項は、請求先アカウント、更新、返金資格、および税金に関する重要な情報を提供します。

1. 必要とされる銀行の譲渡の詳細を確認します。 

1. **選択した購入** 区域で、Microsoft および特定の通貨料金による金額を確認します。 銀行口座からその特定の通貨でその金額を転送する必要があります。

1. 必要な情報がすべてそろったら、[Submit] \ ( **送信**\) を選択します。

   > [!NOTE]
   > 注文 **番号** と **MPN ID** は、注文が正常に **送信** されるとすぐに表示されます。 この情報をメモしておいてください。 銀行の譲渡を完了するには、後で必要になります。

1. 送信が成功したことを示す確認ページが表示されます。 銀行とのネットワーク転送を完了したら、このページに戻って支払いの証明をアップロードする必要があります。

   :::image type="content" source="images/offers/5-mpn-competency-payment-thank-you-confirmation.png" alt-text="送信の確認ページ。" border="false":::

1. 銀行口座からのネットワーク転送を完了します。 次のことを確認してください。

   - ワイヤ転送参照の詳細には、 **注文番号** と **MPN ID** の両方を含めます。

   - (前にメモした通貨で) 支払額を前のページの [ **ワイヤー転送命令** ] 領域に表示されている Microsoft bank アカウントに転送します。

1. 銀行が取引を確認したら、銀行の確認の写真を証明書として取得します。 銀行の確認用のプルーフドキュメントに次の詳細が表示されていることを確認します。 
   - 回線転送の支払いが発生した日付
   - ネットワーク転送の受信に使用された Microsoft 銀行口座
   - 適切な通貨で表示される転送金額
   - 注文番号と MPN ID 番号

1. 電信送金の支払いドキュメントをアップロードする手順については、次のセクションを参照してください。

### <a name="upload-proof-of-wire-transfer-payment"></a>電信送金の支払いをアップロードする

電子送金転送によってシルバーまたはゴールドのメンバーシップ料金を支払う場合は、パートナーセンターに支払いの証明をアップロードする必要があります。 これを行うには、次の手順を実行します。

1. パートナーセンターの [ダッシュボード](https://partner.microsoft.com/dashboard)に戻り、左側のナビゲーションメニューから [ **メンバーシップの提供** ] オプションを選択します。

1. 購入したプランの一覧で、特定のプランを見つけます。 次に、そのプランの [アクション] 列で、[ **Upload 支払いの証明**] を選択します。

   :::image type="content" source="images/offers/6-mpn-competency-payment-show-purchased-offers.png" alt-text="購入したプランごとに選択可能なオプション、アップロードの支払いを示すリストが表示されます。" border="false":::

1. [詳細] ページで、[ **支払いの証明** ] フィールドを見つけます。 [ **参照** ] を選択して、証明書を検索し、パートナーセンターにアップロードします。

   :::image type="content" source="images/offers/7-mpn-competency-payment-upload-proof.png" alt-text="購入したプランごとに、選択可能なオプションであることを証明する証明をアップロードするためのフィールドが表示された [詳細] ページ。" border="false":::

1. 最近の電子ファンド転送取引に関連するその他のフィールドの情報を入力します。たとえば、 **支払い日**、* 支払先口座番号 * *、 **transaction/Ref no** などです。

  要求が正常に完了したことを示す緑色のメッセージが表示されることがあります。

支払いが正常に完了した後、通常、Microsoft の銀行口座に達した後、購入は 48 ~ 72 時間の間にアクティブとして表示されます。 ただし、完全なプロセスを完了するまでに最大 **1 週間** かかることがあります。 

ネットワーク転送の参照の詳細 (注文番号と MPN ID など) に必要な情報を提供し、パートナーセンターに支払いの証明書をアップロードすると、支払いを早くアクティブ化できる場合があります。

お客様の注文作成日から30日以内に支払い手続きが完全に完了しなかった場合、注文の状態は " **期限切れ**" に変わります。 注文が **期限切れ** になった場合は、パートナーセンターから直接新しい注文を作成する必要があります。もう一度支払いを送信する必要はありません。 

## <a name="switch-from-electronic-bank-transfer-payment-to-credit-card-payment-method"></a>電子銀行転送支払いからクレジットカードの支払い方法への切り替え

後で、シルバーまたはゴールドコンピテンシーメンバーシップに関連付けられている料金の支払い方法を変更することができます。 たとえば、電子銀行の譲渡ではなくクレジットカードの支払いを使用する場合があります。 または、unactioned の電子銀行の転送順序がシステムに既に作成されている可能性があります。

支払い設定を既存の注文に変更するには、最初に注文を取り消してから、希望する支払い方法を使用して再購入する必要があります。

次の手順を実行して、既存の unactioned 電子銀行転送を取り消し、支払い設定を変更します。

1. パートナー センター [ダッシュボード](https://partner.microsoft.com/dashboard)にサインインします。

2. 左側のナビゲーションメニューから [ **メンバーシップの提供** 物] を選択します。

3. 購入したプランの一覧で、キャンセルする特定のプランを見つけます。 次に、そのオファーの [アクション] 列で、[ **注文の取り消し**] を選択します。

   :::image type="content" source="images/offers/8-mpn-competency-payment-cancel-order.png" alt-text="購入したメンバーシップの一覧には、[キャンセル順序] オプションが表示されます。" border="false":::

4. 残りの手順に従って、注文を取り消します。 

5. 前のセクションを参照して、新しい支払い設定を使用して新しい注文を作成します。これは、シルバーまたは gold のいずれかのメンバーシッププランに基づいています。

## <a name="apply-upgrade-discount-when-moving-from-silver-to-gold"></a>シルバーからゴールドへの移行時にアップグレード割引を適用する

組織がアクティブなシルバーコンピテンシーを持っていて、関連する gold コンピテンシーの資格要件を満たしている場合は、割引の対象として、シルバーからゴールドにアップグレードすることができます。 Gold コンピテンシーの価格に対して25% の割引が提供されます。

購入日から1年間、新しいゴールド記念日がスケジュールされます。 すべてのゴールド特典を利用できます。

> [!NOTE]
> このアップグレード割引にはいくつかの例外があります。 シルバーメンバーシップオファーを使用していて、25% の Gold メンバーシップ割引を使用して Gold メンバーシッププランにアップグレードした場合、シルバー返金の対象になりません。 これは、シルバーのメンバーシップを30日以内に支払っていて、関連する特典をまだ使用していない場合でも当てはまります。

25% の割引が適用されている場所を確認するには、次の手順を実行します。

1. パートナー センター [ダッシュボード](https://partner.microsoft.com/dashboard)にサインインします。

2. 左側のナビゲーションメニューから [ **メンバーシップの提供** 物] を選択します。

3. [購入可能なプラン] で、金色のメンバーシップタイルを探します。 以前にシルバーのメンバーシップを取得して支払ったことがある場合は、25% の割引を反映するように Gold メンバーシップの金額が調整されます。

   :::image type="content" source="images/offers/9-mpn-competency-silver-to-gold-discount.png" alt-text="購入したメンバーシップの特典には、シルバーからゴールドの25% の割引付きゴールドタイルプランが表示されます。" border="false":::

4. 割引を反映した金色のメンバーシップタイル内で [ **サブスクライブ** ] を選択します。 購入の詳細ページには、選択した購入の割引も表示されます。

   :::image type="content" source="images/offers/10-mpn-competency-silver-to-gold-discount-2.png" alt-text="[詳細] ページには、25% の割引料金が表示されます。" border="false":::

5. [ **次** へ] を選択して、このプランの購入を完了し、割引を受けます。
