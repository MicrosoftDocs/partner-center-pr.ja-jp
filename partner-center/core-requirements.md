# <a name="incentives-co-op-claims-core-requirements"></a>協力インセンティブの申請の主要な要件

**適用対象**

- パートナー センター

**適切なロール**

- インセンティブ管理者
- インセンティブ ユーザー

インセンティブの申請対象となるアクティビティには、種類ごとに主要な要件が設けられています。 お客様は、これらの要件が満たされていることを実行証明ドキュメントで示す必要があります。 一部のアクティビティには、特定のコミュニケーションの種類とメトリックも必要です。

次の表に、同意された各種アクティビティの主要な要件の概要を示します。 

## <a name="demand-generation"></a>需要創出

### <a name="print-advertising"></a>印刷物による広告

| カテゴリ | 実行証明    |
| ------ | ----------- |
| 主要な要件    | 会社のブランドと連絡先情報<br><br>マイクロソフトのコンテンツ |
| コミュニケーションの要件 | プログラム固有の主要な要件を示す最終的なコミュニケーション媒体 (広告に基づく) のコピー |
| メトリック     | 推定される顧客 (またはパートナー) の範囲 |
|対象となる経費    | サード パーティのメディア プレースメント<br><br>印刷物<br><br>料金 - 業者: 業者の料金を含むマーケティングおよびクリエイティブ サービス  |
