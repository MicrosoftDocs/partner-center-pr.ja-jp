# <a name="bulk-operations-for-co-sell-opportunities-using-comma-separated-valuecsv-files"></a>コンマ区切り値 (CSV) ファイルを使用した共同販売機会の一括操作

**適切なロール**

- 紹介管理者
- 紹介ユーザー

パートナーセンターで一括操作を行うと、共同販売機会データをエクスポートおよびインポートすることができます。 [共同販売案件] ページに移動して、ページタイトルバナーの右上にある [インポート/エクスポート] リンクを見つけます。 **参照管理者** と **参照ユーザー** の両方のアクセス許可を持つユーザーは、この機能を使用できます。

> [!IMPORTANT]
> 一括インポートによって実行される作成/更新アクションは、元に戻すことができません。 大量のレコードを変更または作成する場合は、注意してください。 取引を作成した後に変更できるのは、フィールドのサブセットだけです。 **拒否、期限切れ、受注、紛失などのターミナル状態になると、アクションは許可されません。**

## <a name="exporting-co-sell-opportunities"></a>共同販売機会のエクスポート

エクスポート機能の詳細を以下に示します。

- **最大で5000のレコード** をエクスポートするには、[エクスポート] ボタンをクリックします。
- ダウンロードされる取引は、お客様のアクセスレベルに基づいて作成されます。 紹介管理者と紹介ユーザーは、そのスコープに基づいて異なる結果を得られ、取引にチームメンバーとして含めることができます。 [参照権限](permissions-overview.md#manage-referrals)の詳細については、こちらを参照してください。
- Export 関数は、[共同販売機会] ページの現在のタブと、適用されているフィルターを考慮に取ります。
- 適用されたフィルターに基づいて、すべてのデータを含む CSV ファイルが生成されます。
- レコードのダウンロードには最大1分かかる場合があります。
- ダウンロード操作が完了するまで待つ必要はありません。 パートナーセンター内の他のページに移動した場合でも、エクスポート機能が完了するとすぐにファイルがダウンロードされます。
- ダウンロードしたファイルを再利用して案件の詳細を変更し、アップロードしてレコードを更新することができます。

## <a name="importing-co-sell-opportunities"></a>共同販売機会のインポート

- インポート機能を使用して、 **最大1000レコード** を作成または更新できます。
- テンプレートを最初から作成するには、パートナーセンターの [インポート] ページからテンプレートをダウンロードします。
- エクスポート機能を使用して、既存のレコードをダウンロードし、更新することもできます。
- ファイルに1000個を超えるレコードがある場合は、処理できません。
- ファイルが処理されると、作成、更新、および処理されていない参照の数を含む概要が、最後のプロセスファイルカードに表示されます。
- 処理されたレコードの詳細をダウンロードし、エラーを修正し、同じファイルをアップロードして、前回の実行で失敗したレコードを作成または更新することができます。 **前回の実行で失敗した修正されたレコードをアップロードする前に、ファイルからすべての成功したレコードを削除します。**
- ソリューションを追加するには、ソリューション1の横に列名を追加し、ソリューション X として列名を使用します。 X は、取引先のソリューションの番号を表します。 たとえば、ソリューション2、ソリューション3です。
- 処理には、最大50のソリューションを追加できます。
- チームメンバーを追加するには、チームメンバー1の横に列を追加し、チームメンバー X として列名を使用します。 X は、取引先のチームメンバーの番号を表します。 たとえば、チームメンバー2、チームメンバー3です。
- 1つの案件に最大50のチームメンバーを追加することができます。

> [!NOTE]
> 処理が完了するまで待つ必要はありません。 処理が完了すると、最後に処理されたファイルの詳細をダウンロードできるようになります。 **1000レコードを含むファイルをアップロードする場合、最大で10分かかることがあります。**
