# <a name="create-and-manage-incentives-payout-and-tax-profiles-in-partner-center"></a>パートナーセンターでインセンティブの支払いと税金のプロファイルを作成および管理する

**適切なロール**: インセンティブ admin |アカウント管理者 |全体管理者

特定の MPN の場所についてインセンティブ プログラムの支払いを受けるには、有効な支払いおよび税務プロファイルを、プログラムおよび MPN の場所に事前に関連付けて、登録を完了しておく必要があります。 支払いの発行には、この支払いおよび税務プロファイルが使用されます。 インセンティブ プログラムのルールによっては、支払いに対して銀行の電子送信やクレジット ノートを使用できる場合があります。 

## <a name="roles-currencies-and-multiple-microsoft-incentive-programs"></a>役割、通貨、および複数の Microsoft インセンティブプログラム

支払いと税金のプロファイルを開始する前に、以下の情報を理解しておくことが重要です。

### <a name="roles-and-permissions"></a>ロールとアクセス許可

インセンティブ支払いのために銀行および税金の情報を入力するには、インセンティブ管理者である必要があります。 MPN/アカウント管理者の場合は、自分または同僚を割り当てて、インセンティブ管理者にすることができます。

インセンティブ管理者のアクセス許可を要求する必要がある場合は、MPN 管理者または全体管理者にお問い合わせください。 [パートナーセンターのダッシュボード](https://partner.microsoft.com/dashboard/)にサインインすることで、社内のユーザーがこれらのロールを持っていることを確認できます。 右上にある [ **設定** ] アイコンから、[ **ユーザー管理** ] を選択し、グローバル管理者に対してフィルター処理を行います。

インセンティブのユーザーは、インセンティブの利益および支払いの詳細とレポートを表示できますが、銀行や税金の詳細を編集することはできません。

### <a name="choose-your-disbursement-currency"></a>出金通貨の選択

インセンティブ支払いは、支払いプロファイルの設定時に選択した通貨で行われます。 支払いは、Microsoft が毎月設定した換算レートを使用して計算されます。 通貨が選択されているため、値の変更について責任があります。

### <a name="using-different-profiles-for-different-microsoft-programs"></a>さまざまな Microsoft プログラムに異なるプロファイルを使用する

企業が複数のインセンティブプログラムに登録されている場合は、そのすべてに同じ支払いアカウントを使用することも、異なるプログラムに対して異なる支払いアカウントを使用することもできます。

## <a name="create-and-manage-payout-and-tax-profiles-in-partner-center"></a>パートナー センターで支払いと税金のプロファイルを作成および管理する

以下のセクションでは、パートナーセンターで支払いと税金のプロファイルを作成および管理するプロセスについて説明します。

>[!IMPORTANT]
>パートナーセンターで支払いと税金のプロファイルを作成または管理するには、インセンティブ管理者である必要があります。 インセンティブの役割は、各インセンティブプログラムの下にある各 MPN の場所に割り当てる必要があります。 ユーザー アカウントにインセンティブ管理者を追加する方法の詳細については、「パートナー センターアカウントの作成 [」を参照してください](create-user-accounts-and-set-permissions.md)。

   - ドメインでしかサインインできない場合で、追加のドメインが必要な場合は、アカウント管理者に問い合わせ、Azure AD アカウントにドメインを @onmicrosoft.com 追加してください。
   - [仕事用または学校アカウント]または [個人用アカウント] を選択するように求めるメッセージが表示されたら、[仕事用または **学校アカウント] を選択します**。

2. [ **アカウント設定] を選択** し、[ **支払いと税金] セクションを展開** し、[支払いと税金プロファイルの割り当 **て] を選択します**。 
   
   プログラムの一覧が表示されます。 プログラムの横にある矢印を選択して、プロファイルの詳細を表示します。 

3. [税 **プロファイル] ドロップダウン** メニューで、必要な税プロファイルを選択するか、新しいプロファイルを作成するオプションを選択します。 新しいプロファイルを作成するオプションを選択すると、適切にリダイレクトされます。  ポップアップ **ウィンドウで** [続行] を選択します。 新しい税プロファイルを作成するプロセスを以下に示します。

   - 支払い方法として **[電子銀行** 振り込み] を選択した場合は、目的の支払いプロファイルを選択するか、新しいプロファイルを作成するオプションを選択します。 新しいプロファイルを作成するオプションを選択すると、適切にリダイレクトされます。 ポップアップウィンドウで [続行] を選択します。 新しい支払いプロファイルを作成するプロセスを次に示します。

       場所 MPN および支払い方法に関連付けられている CSP テナント ID の詳細を確認して確認するために **必要な** 確認を選択し、クレジットメモの支払いを受け取ります。 [ **クレジットメモの詳細** ] ダイアログボックスで、指定された CSP テナント ID と詳細が正しいことを確認します。 複数のテナント ID が表示される場合は、支払いを受け取る CSP テナント ID を慎重に選択してください。 次に、[確認] を選択して会社の詳細が正しいことと、選択した CSP テナント ID に対してインセンティブ支払いを行う必要があることを **確認** します。
 
      状態が [ **確認** 済み] になっている場合は、CSP テナント ID の割り当てが完了しているため、これ以上の操作は必要ありません。 [確認] を選択して、割り当ての詳細を確認することもできます。
   
      パートナーが明示的に税の適用を要求する必要がある国では、インセンティブ プログラムと場所 MPN の税プロファイル セクションの税プロファイルの横に税の除外を適用するオプションがあります。 このボックスをオンにすると、インセンティブクレジット ノートに税の除外特典が適用されます。 
   
      現在、LRD クレジット ノートの支払い方法は、Microsoft コマース インセンティブ プログラムのオーストラリア、ニュージーランド、カナダのパートナーでのみ利用できます。 MCI プログラムに登録されているこれら 3 つの国の直接請求パートナーまたは間接プロバイダーで、利用可能な支払い方法として LRD クレジット ノートが表示されていない場合は、テナント ID が関連するパートナー MPN の場所アカウントに関連付けられているか確認します。 詳細については、組織プロファイルを更新 [する方法に関するページを参照してください](update-your-partner-profile.md)。

## <a name="set-up-a-default-bank-profile"></a>既定の銀行プロファイルを設定する

既定の銀行プロファイルを設定し、MPN の場所に割り当てできます。 これらの既定のプロファイルは、その MPN の場所に対する後続の登録に Microsoft によって使用されます。 

1. ダッシュボード に[サインインパートナー センター、](https://partner.microsoft.com/dashboard/)歯車アイコンを選択して [設定] メニュー **を開**   きます。 

2. [ **アカウント設定] を** 選択し、[ **支払いと税金] セクションを** 展開し、[支払いプロファイルと税金プロファイル **] を選択します**。 

3. [支払 **いプロファイル] セクションの [** 既定のプロファイル **の管理] を** 選択します。 

4. 既定の銀行プロファイルを作成するには、[既定の銀行 **プロファイルを追加する] を選択します**。 

5. 会社の利用可能な銀行プロファイルの一覧から銀行プロファイルを選択し、この銀行プロファイルで使用する通貨を選択してから、この既定のプロファイルを適用する MPN の場所の一覧を選択します。

6. 選択 **が完了** したら、[完了] を選択します。 [完了] ボタンは、すべての必須フィールドが完了するまでクリックできません。 

>[!NOTE]
>同じ銀行と通貨のペアを複数の場所に適用できます。 場所 MPN に1回既定のプロファイルと通貨の組み合わせが割り当てられている場合は、今後の既定のプロファイルの割り当ての [場所] ドロップダウンに表示されなくなります。 既定の選択が削除された場合、その後の既定のプロファイルの割り当てについては、場所 MPN が再表示されます。 各銀行のプロファイルと通貨の組み合わせは、一意の編集可能な行として追加されます。

7. 必要な変更がすべて追加されたら、[ **保存**] を選択します。  

## <a name="create-your-bank-profile"></a>銀行のプロファイルを作成する

銀行のプロファイルは、会社レベルで作成されます。 これにより、企業内の複数の MPN ID およびインセンティブプログラムに対して1つの銀行プロファイルを割り当てることができます。 異なる銀行や税金のルールが適用される可能性があるため、銀行のプロファイルを異なる国に適用するときに例外が発生する場合があります。

>[!NOTE]
>次のページでは、アスタリスクの付いたフィールドが必要です。 フィールドの内容がわからない場合は、情報アイコンを選択します。 

1. [ **詳細** ] ページで、次のフィールドを入力します。 [ **プロファイル名]:** この支払いプロファイルを識別する一意の名前を入力します。
    **銀行口座の場所:** 会社の銀行がある国。
    **支払い方法:** パートナーセンターのお支払い方法として、電子銀行の譲渡をお勧めします。

2. **[次へ]** を選択します。

3. [ **銀行口座** ] ページで、情報を入力します。 このページに表示されるフィールドは、国によって異なります。 

4. **[次へ]** を選択します。

5. [ **受益者** ] ページで、適切な情報を入力します。 受益者は、お客様がアカウントについて話し合う必要がある場合に、銀行が連絡する会社の人物です。

6. フィールドが完了したら、[ **完了**] を選択し、[ **確認** ] を選択して、銀行プロファイルを作成します。

[ **支払いと税金のプロファイル** ] ページにリダイレクトされます。 新しいプロファイルの状態には、検証が完了するまで、 **保留中の Microsoft 検証** が反映されます。 このプロセスには最大 48 時間かかる場合があります。 検証が完了すると、プロファイルの状態に [承認済み] または [ **アクションが必要]** **が反映されます**。 [ **アクションが必要] の** 場合は、必要な情報を指定して上記の手順を繰り返します。 

## <a name="create-your-tax-profile"></a>税プロファイルを作成する

組織に必要な税金情報を Microsoft に提供するには、次の手順に従います。 このセクションのページは動的であり、国または地域によって異なります。 適切な税金情報の特定に関するヘルプが必要な場合は、お客様の国の適切な政府機関に問い合わせください。

米国内のパートナー企業の場合、W8 または W9 フォームの入力に関する情報が必要な場合は、次のアドレスを使用して IRS サイトに移動します。

- [http://www.irs.gov/pub/irs-pdf/iw8.pdf](http://www.irs.gov/pub/irs-pdf/iw8.pdf)
- [http://www.irs.gov/pub/irs-pdf/iw9.pdf](http://www.irs.gov/pub/irs-pdf/iw9.pdf)

>[!IMPORTANT]
> 会社の詳細のみを入力します。 個人情報を入力しない。

1. [ビジネス プロファイル **] ページで** 、必要なフィールドに入力し、[次へ] を **選択します**。 

2. [セットアップ **] ページ** で、会社に適用するオプションを選択します。

   - 会社が会社に組み込まれる場合は左側のオプションを米国、このプロファイルが個人向けである場合は選択します。
   - 会社が 米国 の外部に組み込まれる場合は、右側のオプションを選択し、一覧から国/地域を選択します。

3. **[次へ]** を選択します。 

4. [税の **状態] ページ** で、必要な情報を入力し、[次へ] を **選択します**。 このページのフィールドは国によって異なります。 詳細を確認します。 

5. [追加の **ドキュメント] ページで** 、必要なフィールドを選択し、[次へ] を **選択します**。 

6. [ **参照]** を選択して、国または地域で必要なドキュメントをアップロードします。 ドキュメント名が表示された場合は、 [アップロード] を **選択します**。 

7. ドキュメントを削除する必要がある場合は、[ **削除**] を選択します。

8. 保存して続行するには、[ **完了**] を選択します。

9. ポップアップメッセージで [ **確認** ] を選択します。 [ **支払いと税金の設定** ] ページに戻ります。

1. [ **税プロファイル**] を選択します。

1. 列の **有効期限** を確認し、有効期限が切れたか、期限切れになる税金プロファイルに移動します。

1. **[編集]** を選択します。

1. [税フォーム] セクションで、新しい詳細を入力して、税フォームを更新します。
